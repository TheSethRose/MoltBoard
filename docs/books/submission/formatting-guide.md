## Standard Manuscript Format Checklist

| Element | Specification | Status |
|---------|---------------|--------|
| **Font** | Times New Roman, 12pt | ✅ |
| **Margins** | 1" top/bottom, 1.25" left/right | ✅ |
| **Line Spacing** | Double-space all text | ✅ |
| **Alignment** | Left-aligned (ragged right) | ✅ |
| **Paragraph Indentation** | 0.5" first line (no extra spacing) | ✅ |
| **Chapter Starts** | Begin 1/3 down page, skip 3-4 lines | ✅ |
| **Scene Breaks** | Centered # symbol (#) or blank line | ✅ |
| **Page Numbers** | Top right, starting page 2 | ✅ |
| **Header** | Author Last Name / TITLE / Page # | Optional |
| **Sentence Spacing** | Single space after periods | ✅ |
| **Dialogue** | Standard quotation marks, new speaker = new paragraph | ✅ |

## File Preparation Instructions

### For Word Documents (.docx):

1. **Page Setup:**
   - Margins: 1" top/bottom, 1.25" left/right
   - Paper: US Letter (8.5" x 11")

2. **Font:**
   - Times New Roman, 12pt
   - No styling changes within text

3. **Paragraph Settings:**
   - Line spacing: Double (2.0)
   -Spacing after: 0pt
   - Indentation: 0.5" first line

4. **Save as:**
   - .docx for agents who request it
   - .rtf for universal compatibility

### For PDF Submissions:

1. Use "Print to PDF" from your word processor
2. Ensure fonts are embedded
3. No security restrictions on the file

### For Plain Text Submissions:

1. Remove all formatting
2. Use standard ASCII characters
3. Mark chapter breaks with: CHAPTER X
